# JavaScript Calculator

A web calculator that does basic math operations.

## Features

- Basic math operations
  - Add
  - Subtract
  - Multiply
  - Divide
- Evaluate single pair of numbers at a time
- Negative number
- Decimal point
- Backspace
- All clear
- Keyboard support
