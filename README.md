# JavaScript Calculator

A web calculator that does basic math operations.

## Features

- Does basic math operations
  - Add
  - Subtract
  - Multiply
  - Divide
- Only evaluate single pair of numbers at a time
- Supports Negative number
- Supports Decimal point
- Supports backspace
- Supports clear
- Keyboard support
